# OpenAI scripts

Set `ENV["OPENAI_ACCESS_TOKEN"]`

```
Usage: openai-edit [options]
    -m STR, --model STR              model to use
    -i STR, --instruction STR        tells the model how to edit
    -n INT                           number of edits
    -t INT                           --temperature INT
    -p INT                           --top_p INT
    -d, --debug                      Print request data
    -v, --version                    Show version
    -h, --help                       Show help
```
